# StyleSync Frontend

Frontend dashboard for StyleSync — interactive UI for analyzing websites, editing tokens, previewing styles, locking tokens, and exporting CSS.

## Tech Stack

* React.js
* Zustand
* Tailwind CSS
* Axios
* Vite

## Setup Instructions

1. Clone repository

2. Install dependencies:
   npm install

3. Start frontend:
   npm run dev

## Features

* URL analysis
* Design token extraction
* Live preview
* Token editing
* Token locking
* Version history
* CSS export
